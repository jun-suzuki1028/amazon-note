# Note特化「過去の自分」ペルソナテンプレート

**戦略方針**: 外部データに依存しない、内省的・実体験ベースのペルソナ設定手法

## 🎯 **「過去の自分」手法の強み**

### note特有の制約への対応
- **Google Analytics不可** → 内面的経験をデータとして活用
- **客観データ不足** → 過去の自分のリアルな悩みを基盤とする
- **共感の深さ** → 同じ経験をした人に深く刺さる

### ファン化戦略との適合性
- **人柄重視** → noteの「経験」「属人性」文化に完全適合
- **信頼性** → 実体験に基づく説得力と信頼関係
- **継続性** → 一時的な関係ではなく長期的なファン関係

## Claude Codeへの指示プロンプト

```
「過去の自分」手法でnote特化ペルソナを作成してください。

【基本方針】
1. **実体験ベース**: 想像ではなく、過去に実際に体験した悩み・感情を基盤とする
2. **時系列的な深さ**: 「あの頃の自分」の状況を具体的に思い出す
3. **情緒的なリアリティ**: 当時のモヤモヤ、不安、期待をリアルに表現
4. **解決プロセスの記録**: どうやって問題を解決したかの具体的なステップ
5. **現在の视点**: 過去の自分に伝えたいアドバイスや気づき

【重要な注意点】
- 一般的なユーザー像ではなく、「過去の具体的な自分」をベースとする
- 数字や統計ではなく、生の体験と感情を中心とする
- 記事作成時に「あの頃の自分」と対話できるレベルのリアリティ

以下のテンプレートを基に、深く共感できるペルソナを作成してください。
```

## 💡 成功のための重要なポイント

**🎯 「過去の自分」手法の最大の強み**：
- **外部データ不要** → noteの解析制約を逆手に取る
- **深い共感** → 同じ経験をした人に「まさに！」と思わせる
