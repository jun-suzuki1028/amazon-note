# Perplexity Research実行ガイド

Perplexity.aiを使用した効果的なDeepResearchの実行方法を解説します。

---

## 📌 Perplexityとは

Perplexityは、AIを活用したリアルタイム検索エンジンで、以下の特徴があります：

- **最新情報の取得**: リアルタイムWeb検索による最新データ
- **出典明記**: 全ての情報に参照URLを付与
- **対話的調査**: 追加質問で深掘り可能
- **Markdown対応**: 結果を構造化された形式で取得

---

## 🚀 実行準備

### 1. アカウント準備
- [Perplexity.ai](https://www.perplexity.ai/)にアクセス
- 無料版でも十分な調査が可能
- Pro版（月額$20）なら無制限の詳細調査が可能

### 2. 事前準備物
- 完成したペルソナファイル
- 生成されたリサーチプロンプト
- リサーチテンプレート（結果記録用）

### 3. 作業環境
- Perplexityとテキストエディタを並べて表示
- コピー&ペースト用のメモ帳を準備

---

## 📝 実行手順

### Step 1: 基本設定

1. **Focus Mode設定**
   - 「Writing」または「All」モードを選択
   - 日本語での回答を明示的に指定

2. **初回プロンプト**
   ```
   これから[商品カテゴリ]のAmazonランキング記事作成のための
   リサーチを行います。日本市場（Amazon.co.jp）の情報を中心に、
   Markdown形式で回答してください。
   ```

### Step 2: プロンプト実行

#### 2.1 キーワード・市場調査
```
実行時間: 約2分
期待結果: キーワードリスト、市場規模データ
保存項目: 検索キーワード、月間検索量、トレンド
```

#### 2.2 製品詳細調査
```
実行時間: 約3分
期待結果: TOP10-15商品の詳細情報
保存項目: ASIN、価格、スペック、レビュー要約
重要: 複数回に分けて実行も可（5商品ずつ等）
```

#### 2.3 競合分析
```
実行時間: 約2分
期待結果: 上位サイトの分析データ
保存項目: URL、構成、差別化ポイント
```

#### 2.4 レビュー分析
```
実行時間: 約2分
期待結果: ユーザーの声の要約
保存項目: ポジティブ/ネガティブ要因
```

#### 2.5 トレンド分析
```
実行時間: 約1分
期待結果: 最新動向と将来予測
保存項目: 新技術、価格動向、市場予測
```

### Step 3: 追加調査（深掘り）

不足情報がある場合の追加質問例：

```
「先ほどの商品リストの中で、[特定商品]の
[特定の機能]について詳しく教えてください」

「[ペルソナの悩み]を解決する観点から、
これらの商品を再評価してください」
```

## 💡 効果的な活用テクニック

### 1. 段階的な情報収集

**❌ 悪い例**: 全ての情報を1つのプロンプトで要求
```
「[商品カテゴリ]の全商品情報、競合、トレンド、
レビュー、価格を全て教えてください」
```

**✅ 良い例**: 段階的に詳細化
```
1. 「[商品カテゴリ]の売れ筋TOP10を教えて」
2. 「その中で評価4.5以上の商品を詳しく」
3. 「それらの商品の悪いレビューの共通点は？」
```

### 2. 具体的な条件指定

**❌ 曖昧な指定**
```
「良い商品を教えて」
```

**✅ 具体的な指定**
```
「Amazon.co.jpで、レビュー500件以上、
評価4.0以上、3万円以下の商品」
```

### 3. 出力形式の指定

**推奨形式指定**
```
「以下の形式で回答してください：
- Markdown表形式
- 各商品3つのメリット/デメリット
- 日本円での価格表記」
```

## 📊 結果の整理方法

### 1. 即座にコピー&保存

Perplexityの回答は時系列で流れるため：
- 各回答を即座にコピー
- テンプレートに沿って整理
- 出典URLも必ず記録

### 2. 情報の検証

```yaml
必須確認項目:
  - ASINの実在確認
  - 価格の最新性
  - レビュー数の確認
  - 在庫状況
```

### 3. 構造化保存

```markdown
## 製品調査結果
**調査日時**: 2025-09-08 14:00
**使用プロンプト**: [実際のプロンプト]

### 商品1: [商品名]
- **ASIN**: B0XXXXX（要確認）
- **価格**: ¥XX,XXX
- **出典**: [URL]

[詳細情報]
```

## ⚠️ 注意事項

### 1. 情報の信頼性

- **ASINは推定値**: 実際のAmazonで確認必須
- **価格は変動**: 調査時点の価格として記録
- **在庫状況**: リアルタイムでない可能性

### 2. 制限事項

**無料版の制限**:
- 1日の質問回数制限あり
- 詳細な追加質問に制限
- 画像生成機能なし

**対処法**:
- 重要な調査を優先
- 効率的なプロンプト設計
- 必要に応じてPro版検討

### 3. プライバシー配慮

- 個人情報を含むプロンプトは避ける
- 企業秘密情報の入力禁止
- 公開情報のみで調査実施

## 🔧 トラブルシューティング

### Q: 日本の情報が出てこない
**A**: プロンプトに「日本市場」「Amazon.co.jp」を明記

### Q: 情報が古い
**A**: 「2025年最新」「現在の」等の時期指定を追加

### Q: 商品が見つからない
**A**: カテゴリ名を変えて再検索（例：「ゲーミングモニター」→「PCモニター ゲーム用」）

### Q: 回答が長すぎる
**A**: 「簡潔に」「TOP5のみ」等で制限

### Q: Markdown形式にならない
**A**: 「Markdown表形式で」と明示的に指定
