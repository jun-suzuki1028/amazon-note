# ペルソナ検証チェックリスト

作成されたペルソナの品質を体系的に検証し、改善提案を生成するための包括的なチェックリストです。

## 使用方法

1. **完成したペルソナプロフィールを用意**
2. **各セクションの項目を順番にチェック**
3. **×が付いた項目について改善提案を実施**
4. **総合評価で品質レベルを判定**
5. **必要に応じてペルソナを修正・補強**

## 総合評価システム

### 品質レベル判定

**Sランク (90-100点)**: 即座に記事作成に活用可能
- 全項目でA評価
- 実在感が極めて高い
- 商品カテゴリとの関連性が完璧

**Aランク (80-89点)**: 軽微な調整で活用可能
- ほぼ全項目でA評価
- 一部の詳細化が必要
- 大枠は高品質

**Bランク (70-79点)**: 重要項目の改善が必要
- A評価が7割程度
- いくつかの重要項目で問題
- 改善後の再評価推奨

**Cランク (60-69点)**: 大幅な修正が必要
- A評価が6割程度
- 基本設定に問題あり
- 抜本的な見直しが必要

**Dランク (60点未満)**: 作り直し推奨
- A評価が6割未満
- 根本的な問題が多数
- 新規作成を検討

### 評価項目別重要度

| 項目 | 重要度 | 配点 |
|------|--------|------|
| 基本情報の妥当性 | 高 | 20点 |
| ライフスタイルの現実性 | 中 | 15点 |
| 購買行動の論理性 | 高 | 25点 |
| ニーズ・悩みの深度 | 高 | 20点 |
| 情報消費パターン | 中 | 10点 |
| 記事期待値の合理性 | 中 | 10点 |

## 活用ガイドライン

### チェックリスト実施のベストプラクティス

1. **段階的評価**: 一度に全項目をチェックせず、セクション単位で評価
2. **客観的視点**: 作成者以外の視点からも評価
3. **実例との比較**: 実在の人物や事例と比較検討
4. **継続的改善**: 記事作成後の効果測定でペルソナを更新

### よくある問題パターンと対処法

**問題**: 基本設定が理想的すぎる
**対処**: より平均的・一般的な設定に調整

**問題**: 商品への関心が不自然
**対処**: より段階的で自然な興味の発展プロセスを設定

**問題**: 情報収集パターンが現代的でない
**対処**: 最新のメディア利用動向を反映

**問題**: 購買予算が収入と不整合
**対処**: 年収層別の消費データを参考に調整

このチェックリストを活用することで、記事作成に最適化された高品質なペルソナを作成できます。
