# 5. 競合分析プロンプト

## Perplexity実行用プロンプト

```
【競合分析】ワイヤレスイヤホンのランキング記事・レビューサイト分析

Google検索で「ワイヤレスイヤホン おすすめ 2025」「ワイヤレスイヤホン コスパ」で
上位表示される記事について：

1. **上位5サイトの分析**
   - サイト名とURL
   - 記事の更新日
   - 紹介商品数とラインナップ
   - 独自の評価軸

2. **記事構成の特徴**
   - 選定基準
   - 比較方法（表、グラフ等）
   - 用途別の提案有無

3. **差別化可能なポイント**
   - ITエンジニア向けの視点不足
   - マルチユース（仕事・運動・娯楽）の評価不足
   - 実測データの不足

表形式で比較してください。
```

## 実行メモ

- **優先度**: 中（差別化戦略のため）
- **実行時間**: 約2分
- **期待成果**: 競合記事の分析と差別化ポイント
- **保存先**: `research-data.md`の「競合分析結果」セクション
